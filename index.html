Java game development project

Gravity Flip

My first java game development project is about a game I created called gravity flip. During the development, I gained and solidified my knowledge of Object-oriented programming, which included concepts such as Inheritance, Encapsulation, Polymorphism, and Abstraction.

Inheritance was used in my project to reduce the amount of repeated code, for example, both of my enemies (Aliens and Alien Bosses) in the game would have the same bullet detection method, which then reduces the amount of unnecessary code required, resulting in a more efficient program.
